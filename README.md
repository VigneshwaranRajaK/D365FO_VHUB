# Product Management Suite

This repository contains the core modules for our enterprise solutions:

- **Payroll Management** – Automates salary computation, allowances, deductions, and payslip generation.
- **Property Management** – Manages property records, tenants, lease contracts, and billing.
- **Attendance & Leave Manager** – Tracks employee attendance, leaves, and approvals.
- **Analytics & Reports** – Power BI dashboards and SSRS reports for insights.

## Technology Stack

- **Backend / ERP Development**: Visual Studio (X++ / C# for extensions)
- **Database**: SQL Server (SSMS)
- **Reporting**: SSRS, Power BI
- **Low-Code / Automation**: Power Apps, Power Automate
- **Version Control / AI Assistance**: GitHub + Copilot

## Modules & Features

### Payroll Management
- Automated salary computation
- Tax and deduction management
- Payslip generation (PDF and Email)
- Integration with Core Banking API for salary disbursement

### Property Management
- Tenant & lease tracking
- Billing and payment reminders
- Lease renewal automation
- Reports for occupancy and revenue

### Attendance & Leave Manager
- Daily attendance tracking
- Leave application, approval workflow
- Delegation management
- Integration with HR and payroll

### Analytics & Reports
- SSRS reports for payroll, property, attendance
- Power BI dashboards for KPIs
- Export to Excel/PDF
